# object-locator
Tensorflow implementation for Weighted Hausdorff Distance: A Loss Function For Object Localization https://arxiv.org/abs/1806.07564


# Installation
```
conda env create -f environment.yml

```
# Training
```
conda activate object-locator-tf
python train.py
```
